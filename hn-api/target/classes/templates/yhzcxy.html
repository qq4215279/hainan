一、       特别提示

     本协议约定海南工会与用户之间关于“海南工会云”服务（以下简称“服务”）的权利义务。“用户”是指注册、登录、使用本服务的个人。本协议可由海南工会随时更新，更新后的协议条款一旦公布即代替原来的协议条款，恕不再另行通知，用户可在本APP中查阅最新版本条款。在修改协议条款后，如果用户不接受修改后的条款，请立即停止使用海南工会云提供的服务，用户继续使用海南工会云提供的服务将被是为接受修改后的协议。

二、       账号注册

1、         用户在使用本服务前需要新注册一个“海南工会云”账号。账号应当使用手机号码注册，请用户使用尚未与“海南工会云”账号绑定的手机号码，以及未被海南工会根据本协议封禁的手机号码注册“海南工会云”账号。海南工会可以根据用户需求或产品需求对账号注册和绑定方式进行变更，而无须事先通知用户。

2、         如果用户注册申请者有被海南工会封禁的先例或涉嫌虚假注册及滥用他人名义注册，及其他不能得到许可的理由，海南工会将拒绝其注册。

3、         鉴于“海南工会云”账号的绑定注册方式，您同意在注册时将允许您的手机号码及手机设备识别码等信息用于注册。

4、         在用户注册及使用本服务时，海南工会需要搜集能识别用户身份的个人信息以便海南工会在必要时联系用户，或为用户提供更好的使用体验。搜集的信息包括但不限于用户的姓名、地址；海南工会同意对这些信息的使用将受限于第三条用户个人隐私信息保护的约束。

三、       账户安全

1、         用户一旦注册成功，成为海南工会云的用户，将得到一个用户名和密码，并有权力使用自己的用户名及密码随时登录海南工会云。

2、         用户对用户名和密码的安全负全部责任，同时对以其用户名进行的所有活动和时间负全责。

3、         用户不得以任何形式擅自转让或授权他人使用自己的海南工会云用户名。

4、         如果用户泄露了密码，有可能导致不利的法律后果，因此不管任何原因导致的用户的密码收到威胁，请立即和海南工会云客服人员取得联系，否则后果自负。
